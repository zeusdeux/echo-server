Echo server

This is an echo http and websocket server.

Other than /hello try any route, query params, HTTP verb with it. For example: https://echo.mudit.xyz/users?sort=asc

You can also connect to using new WebSocket('wss://echo.mudit.xyz') and it'll echo back things to you

This is only an experiment to play with some technologies (node, docker, nginx, letsencrypt) on my part.
